这是我的第一篇博客文章，欢迎访问！

## 关于这个博客

这个博客是基于以下技术栈构建的：

- **Vite** - 极速的构建工具
- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - JavaScript 的超集
- **Markdown** - 轻量级标记语言

## 功能特性

目前博客支持以下功能：

1. 文章列表展示
2. 文章详情页
3. Markdown 内容渲染
4. 响应式设计
5. Frontmatter 元数据解析
6. 标签和分类系统

## 代码示例

```javascript
function greet(name) {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```
