Markdown 是一种轻量级的标记语言，非常适合用来写博客文章。

## 基本语法

### 标题

使用 `#` 符号表示标题，支持 1-6 级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 列表

无序列表使用 `-` 或 `*`：

```markdown
- 项目 1
- 项目 2
- 项目 3
```

有序列表使用数字：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 代码块

使用三个反引号包裹代码，可以指定语言：

```javascript
const message = 'Hello, World!'
console.log(message)
```

### 引用

使用 `>` 符号：

> 这是一段引用文字。

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

## 表格

| 列 1 | 列 2 | 列 3 |
|------|------|------|
| 数据 1 | 数据 2 | 数据 3 |
| 数据 4 | 数据 5 | 数据 6 |

## 分割线

使用三个或更多的 `-`、`*` 或 `_`：

---

***

## 高级技巧

### 转义字符

如果需要显示特殊字符，可以使用反斜杠转义：

```markdown
\*这不是斜体\*
```

### HTML 支持

Markdown 支持 HTML 标签：

```html
<div style="color: red;">红色文字</div>
```

## 总结

掌握这些基本语法，就可以写出漂亮的博客文章了！
